# 🃏 Super Trunfo em C — Nível Mestre

Este projeto é a versão final (nível mestre) do desafio Super Trunfo em C.  
Ele integra menus dinâmicos, comparação de múltiplos atributos, soma de valores e tratamento de empates.

O objetivo foi aplicar de forma avançada estruturas de decisão e boas práticas em C.

---

## 🚀 Funcionalidades

✔ Menu interativo com `switch`  
✔ Escolha de **dois atributos diferentes** para comparação  
✔ Bloqueio de atributos repetidos  
✔ Regra especial para Densidade Demográfica (vence o menor valor)  
✔ Soma dos dois atributos escolhidos  
✔ Tratamento de empate  
✔ Uso do operador ternário  
✔ Código organizado e comentado  

---

## 🧠 Conceitos Aplicados

- `struct`
- `switch`
- `if-else` aninhado
- Operador ternário (`? :`)
- Validação de entrada
- Cálculos com variáveis `int` e `float`
- Organização e legibilidade de código

---

## 🎮 Como Funciona

1. O jogador escolhe o primeiro atributo.
2. O sistema impede que o mesmo atributo seja escolhido novamente.
3. O jogador escolhe o segundo atributo.
4. O programa:
   - Compara os dois atributos individualmente
   - Soma os valores
   - Determina o vencedor com base na soma
   - Exibe o resultado completo
